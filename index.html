Daniel Ding

Computer Engineering Student & AI Research Assistant

I'm researching AI-driven software engineering and automated program repair at the University of British Columbia. My work focuses on understanding how large language models and coding agents can improve software development through better reasoning, context understanding, and multi-step coordination. I'm particularly interested in how these systems handle complex, multi-location bugs and how we can build more reliable AI tools for developers.
